# Analiza i klasifikacija recenzija

## Opis projekta
Ovaj projekat je realizovan u programskom jeziku Clojure i bavi se 
analizom i klasifikacijom recenzija na lažne i prave korišćenjem
algoritma logističke regresije.

## Struktura projekta
- data-source.clj – učitavanje csv fajla,
- preprocess.clj – priprema podataka za analizu,
- analytics.clj – analiza podataka,
- logistic-regression.clj – logistička regresija,
- core.clj – pokretanje aplikacije.
